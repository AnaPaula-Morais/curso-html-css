O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação.

De acordo com página MDN blackquote:

O elemento quote - <q> - é Usado para citações curtas que não precisam de parágrafo ou quebra de linha. -- MDN q página.

Usamos HTML pata estruturar nossos elementos na página.
